A growing toolbox of terminal scripts for Arch Linux users.
This repository is a collection of various bash scripts designed to make your terminal experience more efficient and powerful. Think of it as an ever-growing toolbox—each script is a tool to help you automate, customize, or streamline tasks in Arch Linux.
Features

Diverse Scripts: From system maintenance to custom utilities, find scripts for a wide range of use cases.
Arch Linux Focused: Tailored for Arch Linux, but many scripts are adaptable to other distributions.
Community Driven: Open to contributions and suggestions. Got a script to share? PRs are welcome!
Usage

Clone the repository:


git clone https://github.com/kappel79/bash_scripts_arch.git
